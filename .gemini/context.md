# Project Context: K-Stock-AutoTrader

## 1. 프로젝트 목표
Python을 이용한 한국 주식 시장 자동 매매 시스템 구축. TDD를 기반으로 안정적이고 확장 가능한 코드 구조를 지향한다.

## 2. 개발 로드맵 및 마일스톤

### Milestone 1: 인프라 및 인증 시스템 구축 (현재 단계)
- [ ] 개발 환경 설정 (pytest, 필수 라이브러리 설치)
- [ ] 한국투자증권 KIS API 인증 모듈 구현 (OAuth2.0)
- [ ] 토큰 발급 및 자동 갱신 로직 (TDD 적용)
- [ ] 계좌 기본 정보(잔고, 보유 종목) 조회 기능

### Milestone 2: 시장 데이터 수집 및 분석
- [ ] 종목별 실시간 현재가 수집 모듈
- [ ] 과거 차트 데이터(일/분봉) 수집 및 데이터프레임 변환
- [ ] 기술적 지표(이동평균선, RSI 등) 계산 엔진

### Milestone 3: 주문 집행 및 리스크 관리
- [ ] 매수/매도 주문 실행 기능 (시장가/지정가)
- [ ] 주문 결과 확인 및 상태 관리
- [ ] 손절(Stop-loss) 및 익절(Take-profit) 감시 로직

### Milestone 4: 투자 전략 및 자동 매매 루프
- [ ] 전략 패턴(Strategy Pattern) 기반 투자 로직 분리
- [ ] 메인 자동 매매 루프(Scheduler) 구현
- [ ] 장 시작/종료 시간에 따른 자동 실행 및 정지 로직

### Milestone 5: 모니터링 및 로깅
- [ ] 텔레그램/슬랙 알림 연동
- [ ] 일자별 수익률 및 매매 내역 기록 (CSV/DB)
- [ ] 예외 처리 및 오류 복구 시스템 강화

## 3. 핵심 규칙
- 모든 기능은 `.gemini/skills/test-driven-development.md`에 따라 TDD로 개발한다.
- 코드 작성 전 반드시 설계 문서(`docs/`)를 업데이트하고 정합성 검증을 거친다.
- 한국투자증권 API 문서를 최우선 참조한다.
